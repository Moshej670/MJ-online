About Me

A journey of curiosity and discovery

Hi, I’m Moshe Janani. My learning journey began in web development—building and maintaining websites, writing custom code, and troubleshooting through hands-on experimentation. This foundation in frontend development naturally expanded into backend programming and creating scripts and applications. Since then, I have immersed myself in research across computational chemistry, chromatin modeling, cancer biology, and neurodegeneration. I’ve worked on simulations, developed computational models, and gained experience with clinical, computational, and wet lab techniques. Beyond the lab, I’ve volunteered in hospitals, served in student government, and contributed to interdisciplinary projects that bridge science, technology, and real-world challenges.

This website is a Github repository I created when I was younger, it contains my achievements, test files, and code I found cool and wanted to share. Additionally, this site is no longer updated regularly. Many functionalities no longer work and those that do are under my admin section. I coded this site myself during my free time. It includes features like responsive design which I learned from Yaakov Chaikin on Coursera. Check out his site at Clearly Decoded.
